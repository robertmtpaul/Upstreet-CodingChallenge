# Upstreet-CodingChallenge

KYC check built for coding challenge calling third-party provider Testing API using a Node.js TypeScript function

### Installation/running
Clone repository, then run: 
```bash
$ yarn
```
NB: * Yarn is being used for scripts in package.json

### Run:
```bash 
$ yarn start
```

### Testing
I used Jest to carry out unit tests, and to run these I used this command:

```bash
$ yarn test
```
